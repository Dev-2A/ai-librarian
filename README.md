# 📚 AI 사서 (AI Librarian)

임베딩 기반 개인 독서 큐레이션 봇  

사용자가 읽은 책의 감상평을 입력하면, 임베딩 모델로 벡터화하여  
취향에 맞는 유사 도서를 추천하는 시스템입니다.

## 기술 스택

- **Backend:** FastAPI + Python 3.11
- **Search Engine:** Elasticsearch 8.x (벡터 검색)
- **Embedding:** BGE-M3
- **Frontend:** React
- **External API:** 알라딘 Open API

## 프로젝트 구조

```text
ai-librarian/
├── backend/          # FastAPI 백엔드
├── frontend/         # React 프론트엔드
├── docker-compose.yml
└── README.md
```
